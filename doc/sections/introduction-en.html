Introduction back to ToC

Corago Semantic Model (Corago SM) ontology was designed to be a natural extension of CIDOC CRM and FRBRoo compared to which it is therefore natively aligned.
Its main function is to integrate and specialize some definitions of concepts defined by the two reference ontologies in order to better represent some of the contents of the Corago archive. The extensions concerned the introduction of new classes, a series of object properties and some data properties.
The ontology is defined using the OWL-DL 1.0 language and includes in its definition the import of the types defined in CIDOC CRM and FRBRoo in the so-called "Erlangen" version.
